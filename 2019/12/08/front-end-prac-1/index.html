мң„ м„ё м–ём–ҙлҠ” мӣ№лёҢлқјмҡ°м Җм—җм„ң кө¬лҸҷлҗҳлҠ” м–ём–ҙмһ…лӢҲлӢӨ. к·ёлһҳм„ң мӣ№н”„лЎ нҠём—”л“ң м–ём–ҙлқјкі  л¶ҖлҰ…лӢҲлӢӨ.

мӣ№лёҢлқјмҡ°м ҖлҠ” м„ңлІ„лЎңл¶Җн„° HTML/CSS/JavaScript мҪ”л“ңлҘј л°ӣм•„м„ң, мқҙлҘј к·ёлһҳн”Ҫм ҒмңјлЎң н•ҙм„қн•ҙм„ң мң м Җм—җкІҢ ліҙм—¬мӨҚлӢҲлӢӨ.

  • HTML (Hyper Text Markup Language) : л¬ём„ңлҘј кө¬мЎ°м ҒмңјлЎң н‘ңнҳ„
  • CSS (Cascasding Style Sheet) : мӣ№нҺҳмқҙм§Җмқҳ к°Ғ мҠӨнғҖмқјмқ„ м •мқҳ
  • JavaScript : мӣ№нҺҳмқҙм§ҖлӮҙ лЎңм§Ғмқ„ кө¬нҳ„

1) н•ң HTMLнҢҢмқјм—җ CSS/JavaScriptлҘј лӘЁл‘җ л„Јмқ„ мҲҳлҸ„ мһҲкө¬мҡ”

cssлҠ” лӮҙмҡ©ліҙлӢӨ м•һм—җ л‘ (мҠӨнғҖмқјмқҙ лЁјм Җ лЎңл”©лҗҳм–ҙм•јн•ҳлҜҖлЎң)

jsлҘј л§Ёл’Өм—җ л‘җлҠ”кІҢ мўӢмқҢ (лЎңм§ҒмӘҪмқҙл©° лЎңл”©мқҙ лҠҗлҰҙк°ҖлҠҘм„ұлҶ’мқҢ)

2) CSS/JavaScript нҢҢмқјмқ„ лі„лҸ„ нҢҢмқјлЎң кҙҖлҰ¬н•ҳмӢӨ мҲҳлҸ„ мһҲмҠөлӢҲлӢӨ.

лі„лҸ„ нҢҢмқј css мҳҲмӢң

  • HTML мқ‘лӢөнҒ¬кё°лҘј мӨ„мқј мҲҳ мһҲмҠөлӢҲлӢӨ.
  • м—¬лҹ¬ лІҲ мғҲлЎңкі м№Ён•ҳлҚ”лқјлҸ„, лёҢлқјмҡ°м Җ мәҗмӢұкё°лҠҘмқ„ нҶөн•ҙ к°ҷмқҖ нҢҢмқјмқ„ м„ңлІ„ лЎңл¶Җн„° лӢӨмӢң мқҪм–ҙл“Өмқҙм§Җ м•ҠмҠөлӢҲлӢӨ.
  • мӣ№нҺҳмқҙм§Җ мқ‘лӢөм„ұмқ„ лҶ’м—¬мӨ„ мҲҳ мһҲмҠөлӢҲлӢӨ.

мӣ№ н”„лЎ нҠём—”л“ңмҷҖ л°ұм—”л“ң

  • мӣ№к°ңл°ңмқҖ нҒ¬кІҢ л°ұм—”л“ңмҷҖ н”„лЎ нҠём—”л“ң к°ңл°ңлЎң лӮҳлҲ м§‘лӢҲлӢӨ. мһҘкі лҠ” л°ұм—”л“ңм—җ мҙҲм җмқҙ л§һм¶° м§„ мӣ№н”„л Ҳмһ„мӣҢнҒ¬мһ…лӢҲлӢӨ.
  • мһҘкі лҘј кіөл¶Җн•ҳмӢӨ л•Ңм—җлҠ” л°ұм—”л“ңм—җ нҸ¬м»ӨмҠӨлҘј л§һм¶°м„ң кіөл¶Җн•ҳмӢңкі , мӣ№н”„лЎ нҠём—”л“ңлҠ” мөңмҶҢнҷ” н•ҳм„ёмҡ”.
    • 2к°Җм§ҖлҘј лӘЁл‘җ н•ңлІҲм—җ мһҳн•  мҲҳлҠ” м—ҶмҠөлӢҲлӢӨ. мҡ°м„ мҲңмң„лҘј л°ұм—”л“ңм—җ лЁјм Җ л‘җм„ёмҡ”. Android/iOS м•ұлҸ„ мқјлӢЁ лҜёлӨ„л‘җм„ёмҡ”.
    • лӢЁ 1лІҲм—җ мҷ„м „н•ң(?) мӣ№м„ңл№„мҠӨлҘј л§Ңл“Ө мҲң м—ҶмҠөлӢҲлӢӨ. м җм§„м ҒмңјлЎң к°ңм„ н•ҙк°Җм„ёмҡ”.

л°ұм—”л“ң к°ңл°ңм–ём–ҙмҷҖ н”„лЎ нҠём—”л“ң к°ңл°ңм–ём–ҙ

  • н”„лЎ нҠём—”л“ң к°ңл°ңм–ём–ҙ (нҒҙлқјмқҙм–ёнҠёлӢЁ мӨ‘ н•ҳлӮҳ1 )
    • лёҢлқјмҡ°м Җ лӢЁм—җм„ң мӢӨн–үмқҙ лҗ©лӢҲлӢӨ.
    • HTML/CSS/JavaScriptмқҳ мЎ°н•© вҖў л°ұм—”л“ң к°ңл°ңм–ём–ҙ (м„ңлІ„ лӢЁ)
  • нҒҙлқјмқҙм–ёнҠё лӢЁмқҳ мҡ”мІӯмқ„ мІҳлҰ¬/мқ‘лӢөл§Ң н•  мҲҳ мһҲмңјл©ҙ лҗ©лӢҲлӢӨ.
    • лӢӨм–‘н•ң м–ём–ҙк°Җ к°ҖлҠҘн•ҳл©°, м—¬лҹ¬ м–ём–ҙ/н”„л Ҳмһ„мӣҢнҒ¬лҘј м„һм–ҙм“°мӢӨ мҲҳлҸ„ мһҲмҠөлӢҲлӢӨ. (Python, NodeJS, Ruby, Java л“ұ)

  • мӣ№мқҖ HTTP(S) н”„лЎңнҶ мҪңлЎң лҸҷмһ‘н•©лӢҲлӢӨ.
  • н•ҳлӮҳмқҳ мҡ”мІӯмқҖ нҒҙлқјмқҙм–ёнҠёк°Җ мӣ№м„ңлІ„лЎң мҡ”мІӯн•ҳл©°, мӣ№м„ңлІ„лҠ” мҡ”мІӯм—җ л§һкІҢ мқ‘лӢөмқ„ н•ҙм•јн•©лӢҲлӢӨ.
    • мқ‘лӢөмқҖ HTML мҪ”л“ң л¬ёмһҗм—ҙ, CSS мҪ”л“ң л¬ёмһҗм—ҙ, JavaScript мҪ”л“ң л¬ёмһҗм—ҙ, Zip, MP4 л“ұ м–ҙл– н•ң нҸ¬л§·мқҙлқјлҸ„ к°ҖлҠҘн•©лӢҲлӢӨ.

мқјл°ҳм Ғмқё мһҘкі  нҺҳмқҙм§Җмқҳ мҳҲ

HTML н…ңн”ҢлҰҝ

мқҙ нҷ”л©ҙмқҙ к·ёл Өм§Җкё°к№Ңм§Җ, мІҳлҰ¬мҲңм„ң

лӢЁм§Җ, н•ҳлӮҳмқҳ HTTP мҡ”мІӯм—җ лҢҖн•ҙ, н•ҳлӮҳмқҳ HTTP мқ‘лӢөмқ„ л°ӣмҠөлӢҲлӢӨ.

  1. лёҢлқјмҡ°м Җм—җм„ң м„ңлІ„лЎң HTTP мҡ”мІӯ

  2. м„ңлІ„м—җм„ңлҠ” н•ҙлӢ№ HTTPмҡ”мІӯм—җ лҢҖн•ң мІҳлҰ¬ мӢӨн–ү : мһҘкі м—җм„ңлҠ” кҙҖл Ё л·° н•ЁмҲҳк°Җ нҳём¶ң

  3. л·° н•ЁмҲҳм—җм„ң лҰ¬н„ҙн•ҙм•јл§Ң л№„лЎңмҶҢ HTTPмқ‘лӢөмқҙ мӢңмһ‘лҗҳл©°, к·ё HTTP мқ‘лӢөмқ„ л°ӣкё° м „к№Ңм§ҖлҠ” н•ҳм–Җ нҷ”л©ҙл§Ң ліҙм—¬м§‘лӢҲлӢӨ. л”°лқјм„ң л·° мІҳлҰ¬мӢңк°„мқҙ кёём–ҙм§ҲмҲҳлЎқ кёҙ нҷ”л©ҙмқҙ ліҙм—¬м§ҖлҠ” мӢңк°„мқҙ кёём–ҙм§‘лӢҲлӢӨ.

  4. лёҢлқјмҡ°м ҖлҠ” м„ңлІ„лЎңл¶Җн„° HTTP л¬ёмһҗм—ҙ мқ‘лӢөмқ„ 1мӨ„м”© н•ҙм„қн•ҳм—¬, к·ёлһҳн”Ҫм ҒмңјлЎң н‘ңнҳ„н•©лӢҲлӢӨ.

м•„м§Ғ, HTML л¬ёмһҗм—ҙ мқ‘лӢөм—җ м¶”к°Җ лҰ¬мҶҢмҠӨ (CSS, JavaScript, Image л“ұ) к°Җ м—ҶмҠөлӢҲлӢӨ.

л§Ңм•Ҫ HTML л¬ёмһҗм—ҙ мқ‘лӢөм—җ м¶”к°Җ лҰ¬мҶҢмҠӨк°Җ м§Җм •лҗҳм–ҙмһҲлӢӨл©ҙ?

HTML л¬ёмһҗм—ҙмқҖ 1мӨ„м”© мІҳлҰ¬лҗҳл©°, мҷёл¶Җ лҰ¬мҶҢмҠӨлҠ” н•ҙлӢ№ лҰ¬мҶҢмҠӨк°Җ лЎңл”©мҷ„лЈҢ/мӢӨн–үлҗ  л•Ңк№Ңм§Җ лҢҖкё°н•©лӢҲлӢӨ.

м•„лһҳ мҳҲмӢңм—җлҠ” link, script к°Җ мҷёл¶ҖмқҙлҜҖлЎң.

  • лҢҖк°ң CSSлҘј HTMLм»Ён…җмё ліҙлӢӨ м•һм—җ мң„м№ҳмӢңнӮӨкі 
    • CSSк°Җ м»Ён…җмё ліҙлӢӨ л’Өм—җ мң„м№ҳн•ңлӢӨл©ҙ, мң м Җм—җкІҢлҠ” мҠӨнғҖмқјмқҙ м Ғмҡ©лҗҳм§Җ м•ҠмқҖ HTMLм»Ён…җмё к°Җ лЁјм Җ л…ём¶ңлҗ  мҲҳ мһҲмҠөлӢҲлӢӨ.
  • JSлҘј HTMLм»Ён…җмё ліҙлӢӨ л’Өм—җ мң„м№ҳмӢңнӮөлӢҲлӢӨ.
    • JSлҠ” мҠӨнғҖмқјм—җ м§Ғм ‘м Ғмқё мҳҒн–Ҙмқ„ лҒјм№ҳм§Җ м•Ҡкё° л•Ңл¬ём—җ, HTMLм»Ён…җмё ліҙлӢӨ лӮҳмӨ‘ м—җ лЎңл”©лҗҳм–ҙлҸ„ лҢҖк°ң кҙңм°®мҠөлӢҲлӢӨ. кјӯ н•„мҡ”н•ң лӘҮлӘҮ JSлҠ” HTMLм»Ён…җмё ліҙлӢӨ м•һм—җ мң„м№ҳмӢңнӮӨкё°лҸ„ н•©лӢҲлӢӨ
