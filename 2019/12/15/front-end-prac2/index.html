  • мһҘкі лҠ” One Project, Multi App кө¬мЎ°
  • н•ң Appмқ„ мң„н•ң static нҢҢмқјмқ„ app/static/appкІҪлЎңм—җ л‘җм„ёмҡ”.
  • н”„лЎңм қнҠё м „л°ҳм ҒмңјлЎң мӮ¬мҡ©лҗҳлҠ” static нҢҢмқјмқ„ settings.STATICFILES_DIRSм—җм„ң м°ёмЎ° н•ҳлҠ” кІҪлЎңм—җ л‘җм„ёмҡ”.

мһҘкі м—җм„ңмқҳ STATIC нҢҢмқј м„ңл№ҷ

мһҘкі мқҳ к°ңл°ңм„ңлІ„м—җм„ң

лёҢлқјмҡ°м Җ мәҗмӢң

лёҢлқјмҡ°м Җ мәҗмӢң кё°к°„мқ„ м„Өм •н•ҙ мЈјл©ҙ(headerм—җм„ңм„Өм •) к·ё кё°к°„ лҸҷм•ҲмқҖ мӣ№лёҢлқјмҡ°м Җк°Җ н•ҙлӢ№ нҢҢмқјмқ„ лӢӨмӢң лӢӨмҡҙл°ӣм§Җ м•Ҡкі  мәҗмӢұлҗң лӮҙмҡ©мқ„ мӮ¬мҡ©н•ҳкё° л•Ңл¬ём—җ нҠёлһҳн”Ҫмқҙ мӨ„м–ҙл“Өкі , мҶҚлҸ„лҸ„ л№Ёлқјм§‘лӢҲлӢӨ.

  • Expires н—ӨлҚ” MDN #doc : л§ҢлЈҢмқјмӢңлҘј м§Җм •

    • Expires: Wed, 21 Oct 2015 07:28:00 GMT

    • мқ‘лӢө лӮҙм—җ вҖңmax-ageвҖқ нҳ№мқҖ вҖңs-max-ageвҖқ directiveлҘј м§ҖлӢҢ CacheControl н—ӨлҚ”к°Җ мЎҙмһ¬н•  кІҪмҡ°, Expires н—ӨлҚ”лҠ” л¬ҙмӢң

  • Cache-Control

мқҙнӣ„м—җ н•ҙлӢ№ нҢҢмқјмқҙ ліҖкІҪлҗҳм—ҲмҠөлӢҲлӢӨ. к·ёлҹ°лҚ°, мғҲлЎңмҡҙ лӮҙмҡ©мқҙ л°ҳмҳҒлҗҳм§Җ м•ҠмҠөлӢҲлӢӨ. ???

  1. мң м ҖлҠ” /blog/ нҺҳмқҙм§Җм—җ л°©л¬ён•ҳл©ҙм„ң лёҢлқјмҡ°м Җм—җ /static/blog/style.css нҢҢмқјмқҙ лӢӨмҡҙлЎңл“ңлҗҳм—ҲмҠөлӢҲлӢӨ. мқҙл•Ң мқҙ нҢҢ мқјмқҙ 24мӢңк°„ лҸҷм•Ҳ лёҢлқјмҡ°м Җ мәҗмӢұмқҙ лҗҳм–ҙмһҲлӢӨкі  мғқк°Ғн•ҙлҙ…мӢңлӢӨ.

  2. к·ёлҹ°лҚ°, к°ңл°ңн•ҳл©ҙм„ң CSSнҢҢмқјмқҙ ліҖкІҪлҗҳм—ҲмҠөлӢҲлӢӨ. нҢҢмқјкІҪлЎңлҠ” л°”лҖҢм§Җ м•Ҡм•ҳмҠөлӢҲлӢӨ. ліҖкІҪлҗң CSSнҢҢмқјмқҙ мң м ҖнҺҳмқҙм§Җм—җ м Ғмҡ©лҗҳкёё мӣҗн•ҳм§Җл§Ң м Ғмҡ©лҗҳм§Җ м•ҠмҠөлӢҲлӢӨ. мәҗмӢұлҗң мқҙм „ нҢҢмқјм—җ кі„мҶҚ м ‘к·јн•ҳкІҢ лҗ©лӢҲлӢӨ.

  3. н•ҙкІ°н•ҳкё°

    вҖӢ л°©лІ•1) н•ҙлӢ№ нҢҢмқјмқҳ мәҗмӢұмқҙ л§ҢлЈҢлҗ  л•Ңк№Ңм§Җ кё°лӢӨлҰҪлӢҲлӢӨ.

    вҖӢ л°©лІ•2) лёҢлқјмҡ°м Җ м„Өм •м—җм„ң мәҗмӢұлҗң лӮҙмҡ©мқ„ мӮӯм ңн•©лӢҲлӢӨ. нҒ¬лЎ¬ лёҢлқјмҡ°м Җм—җм„ңлҠ” вҖңк°•л Ҙ мғҲлЎңкі м№ЁвҖқ 3мқ„ нҶөн•ҙ мҲҳн–ү к°ҖлҠҘ.

    вҖӢ л°©лІ•3) н•ҙлӢ№ STATIC лҰ¬мҶҢмҠӨмқҳ URLмқ„ ліҖкІҪн•©лӢҲлӢӨ.

Tip: л°©лІ•2)к°ңл°ң мӢңм—җ мң мҡ©н•©лӢҲлӢӨ. мңҲлҸ„мҡ° лӢЁм¶•нӮӨ Ctrl+Shift+R, л§Ҙ лӢЁм¶•нӮӨ Command+Shift+R (к°ңл°ңмһҗлҸ„кө¬лқ„м–ҙлҶ“мқҖмғҒнҷ©м—җм„ң)

нҒҙлқјмқҙм–ёнҠёмёЎ мәҗмӢұкіј л№ лҘё м—…лҚ°мқҙнҠёлҘј н• л Өл©ҙ

лҰ¬мҶҢмҠӨмқҳ URLмқ„ ліҖкІҪн•ҳкі  мҪҳн…җмё к°Җ ліҖкІҪлҗ  л•Ңл§ҲлӢӨ мӮ¬мҡ©мһҗк°Җ мғҲ мқ‘лӢөмқ„ лӢӨмҡҙлЎңл“ңн•ҳлҸ„лЎқ н•ҳл©ҙ лҗ©лӢҲлӢӨ.

  1. GETмқёмһҗ л¶ҷмқҙкё° : мӢӨм ң нҢҢмқјлӘ…мқҖ ліҖкІҪн•ҳм§Җ м•Ҡмңјл©ҙм„ң, лёҢлқјмҡ°м Җк°Җ мқём§Җн•ҳлҠ” URLл§Ң ліҖкІҪ

    к°ңл°ң мӢңм—җ мң мҡ©

    лІ„м „мқ„ мҲ«мһҗлЎң л¶ҷмқҙкұ°лӮҳ (м•„лһҳ мҳҲмӢңлҠ” getмқёмһҗлЎң лІ„м „мҲ«мһҗл¶ҷмһ„)(мғҲлЎңмҡҙ urlлЎң мқёмӢқлҗҳлҜҖлЎң лӘЁл“  лҰ¬мҶҢмҠӨ мғҲлЎӯкІҢ лӢӨмҡҙл°ӣкІҢлҗЁ)

    http://localhost:8000/static/main.css?v=1

    лІ„м „мқ„ лӮ м§ңлЎң л¶ҷмқҙкұ°лӮҳ

    http://localhost:8000/static/main.css?v=20180618

    лҚ”лҜёлЎң нҳ„мһ¬мӢңк°Ғмқҳ timestampмқ„ л¶ҷмһ…лӢҲлӢӨ.

    http://localhost:8000/static/main.css?_=1503808011

  2. нҢҢмқјлӘ… ліҖкІҪн•ҳкё°

    м„ңл№„мҠӨ л°°нҸ¬ мӢңм—җ мң мҡ©

м»ӨмҠӨн…Җ н…ңн”ҢлҰҝнғңк·ёлҘј нҶөн•ҙ STATIC URLм—җ лҚ”лҜё GETмқёмһҗ л¶ҷмқҙкё°

н•„мҡ” мң„м№ҳ кІҪлЎң) myapp/templatetags/static_tags.py

static_t нғңк·ё нҷңмҡ©

мң„ лӮҙмҡ©мқҖ м•„лһҳмҷҖ к°ҷмқҙ л ҢлҚ”л§Ғлҗҳл©°, мғҲлЎңкі м№Ён•  л•Ңл§ҲлӢӨ лҚ”лҜё GETмқёмһҗк°’мқҙ ліҖкІҪлҗ©лӢҲлӢӨ. к°ҷмқҖ нҢҢмқјмқҙм§Җл§Ң лёҢлқјмҡ°м Җм—җм„ңлҠ” л§ӨлІҲ мғҲлЎңмҡҙ URLлЎң мқём§Җн•ҳкІҢ лҗ©лӢҲлӢӨ.

/static/blog/style.css?_=1503810446

м•„лһҳмҷҖ к°ҷмқҙ JavaScript/CSS кІҪлЎңм—җ мӮ¬мҡ©н•  мҲҳ мһҲмҠөлӢҲлӢӨ.

лӢӨм–‘н•ң STATIC лҰ¬мҶҢмҠӨ

  • м§Ғм ‘ мғқм„ұ/л“ұлЎқн•ң CSS/JavaScript/Image нҢҢмқјл“Ө
  • мҷёл¶Җ CSS/JavaScript лқјмқҙлёҢлҹ¬лҰ¬
    • CDN (Contents Delivery Network) л°°нҸ¬нҢҗ нҷңмҡ©
    • м§Ғм ‘ лӢӨмҡҙлЎңл“ң&м„ңл№ҷ
    • мһҗл°”мҠӨнҒ¬лҰҪнҠё нҢ©нӮӨм§Җ кҙҖлҰ¬мһҗлҘј нҷңмҡ©н•ҳм—¬ лӢӨмҡҙлЎңл“ң&м„ңл№ҷ

CDN л°°нҸ¬нҢҗ нҷңмҡ©

  • мң лӘ… лқјмқҙлёҢлҹ¬лҰ¬мқј кІҪмҡ°, лҢҖкІҢ CDN л°°нҸ¬нҢҗмқ„ м ңкіө
  • к°ңл°ң мӢңм—җ л№ лҘё м Ғмҡ©мқ„ мң„н•ҙм„ңлҠ” нҺёлҰ¬
  • м•Ҳм •м Ғмқё мӢӨм„ңл№„мҠӨ м ңкіөмқ„ мң„н•ҙм„ңлҠ” лӢӨмҡҙлЎңл“ң&м„ңл№ҷмқ„ м¶”мІң
    • м •м Ғ нҢҢмқј м„ңл№ҷмқ„ вҖңкҙҖлҰ¬н•  мҲҳ м—ҶлҠ” мҷёл¶Җ м„ңл№„мҠӨвҖқм—җ мқҳмЎҙн•  кІҪмҡ°, нҠ№м • мң  м Җмқҳ н•ҙмҷёл§қ м ‘мҶҚмқҙ мӣҗнҷңн•ҳм§Җ м•Ҡкұ°лӮҳ, н•ҙлӢ№ м„ңл№„мҠӨ мһҘм• мқј кІҪмҡ°, мқҳлҸ„м№ҳ м•ҠкІҢ м„ңл№„мҠӨ мқҙмҡ©м—җ м°Ём§Ҳмқҙ л°ңмғқн•ҳкІҢ лҗ©лӢҲлӢӨ.

м§Ғм ‘ лӢӨмҡҙлЎңл“ң&м„ңл№ҷ

  • н”„лЎңм қнҠё м „л°ҳм ҒмңјлЎң мӮ¬мҡ©лҗ  нҢҢмқјл“ӨмқҙлҜҖлЎң, filesystem static finder м—җм„ң м ‘к·јн•ҳлҠ” кІҪлЎңм—җ л„Јм–ҙл‘җкі , лІ„м „кҙҖлҰ¬ лҢҖмғҒм—җлҸ„ м¶”к°Җ
  • вҖңн”„лЎңм қнҠё/static/вҖң кІҪлЎң

мһҗл°”мҠӨнҒ¬лҰҪнҠё нҢ©нӮӨм§Җ кҙҖлҰ¬мһҗлҘј нҷңмҡ©(мўӢм•„)

лёҢлқјмҡ°м Җм—җм„ңлҠ” HTML нҸ¬л§·мқҳ лҚ°мқҙн„°к°Җ н•„мҡ”н•©лӢҲлӢӨ.

1) HTML нҸ¬л§·мқ„ м„ңлІ„м—җм„ң л§Ңл“Өм–ҙм„ң мқ‘лӢөмңјлЎң мЈјкұ°лӮҳ
