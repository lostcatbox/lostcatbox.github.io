  • .is_authenticated - лЎңк·ёмқё м—¬л¶Җ (property) (лЎңк·ёмқёмғҒнғңмӢң True)

  • .is_anonymous - лЎңк·ём•„мӣғ м—¬л¶Җ (property) (лЎңк·ёмқёмғҒнғңмӢң False)

    password н•„л“ңм—җ м ҖмһҘ. saveн•ЁмҲҳлҘј нҳём¶ңн•ҳм§„ м•ҠмҠөлӢҲлӢӨ.(.saveн•ЁмҲҳлҘј нҳём¶ңн•ҙм•ј DBм—җ мҝјлҰ¬ліҙлӮҙм„ң м ҖмһҘ)

  • .check_password(raw_password) - м ҖмһҘлҗң м•”нҳёк°Җ л§һлҠ” м§Җ м—¬л¶Җ

  • .set_unusable_password() - лЎңк·ёмқё л¶Ҳк°Җ м•”нҳёлЎң м„ёнҢ…

    (лҚ°мқҙн„°к°’м—җ вҖҳ!вҖҷк°Җл§Ём•һмқҙлҜҖлЎңм–ҙл–Өк°’мқҙл“  л§Өм№ӯ л¶Ҳк°Җ

лЎңк·ёмқё л¶Ҳк°Җ м•”нҳёлЎң м„ёнҢ…н•ңлӢӨлҠ” кІғмқ„ мқҳлҜё

  • м•”нҳёлҘј нҶөн•ң лЎңк·ёмқёмқ„ н—Ҳмҡ©н•ҳм§Җ м•ҠкІ лӢӨ(нҺҳмқҙмҠӨл¶Ғмқҙлһ‘м—°лҸҷмқҙл©ҙ нҺҳмқҙмҠӨл¶Ғмқ„ кјӯк°”лӢӨмҷҖм•јн•Ё)
  • мҷёл¶Җ м„ңл№„мҠӨ мқёмҰқ(OAuth)м—җ мқҳн•ң мң м Җмқј кІҪмҡ°

UserкҙҖл Ё м¶”к°Җ мҶҚм„ұмқ„ м •мқҳн•ҳкі  мӢ¶лӢӨл©ҙ

  • л°©лІ•2) Custom User Model л§Ңл“Өкё°

    • Custom User Modelмқҳ вҖҳм•ұмқҙлҰ„.лӘЁлҚёлӘ…вҖҷмқ„ settings.AUTH_USER_MODELмңјлЎң м§Җм • (л””нҸҙнҠё: вҖҳauth.UserвҖҷ)

Tip:

  • мқёмҰқ User лӘЁлҚё нҒҙлһҳмҠӨлҠ” ліҖкІҪлҗ  мҲҳ мһҲмңјл©°

    кҙҖкі„ м§Җм •мӢң User лӘЁлҚёнҒҙлһҳмҠӨ м°ёмЎ°лҠ” settings.AUTH_USER_MODEL

  • User лӘЁлҚёнҒҙлһҳмҠӨ нҡҚл“қмқҖ django.contrib.auth.get_user_model

л·°м—җм„ң нҳ„мһ¬ лЎңк·ёмқё мң м Җ нҡҚл“қ

  • л·° FBVм—җм„ң request.user, л·° CBVм—җм„ң self.request.user
    • лЎңк·ёмқё мғҒнғң : settings.AUTH_USER_MODEL нҒҙлһҳмҠӨ мқёмҠӨн„ҙмҠӨ
    • лЎңк·ём•„мӣғ мғҒнғң : django.contrib.auth.models.AnonymousUser (just нҢҢмқҙмҚ¬ н•ЁмҲҳмһ„)
      • лӘЁлҚё мқёмҠӨн„ҙмҠӨк°Җ м•„лӢҲлҜҖлЎң, лӢӨлҘё лӘЁлҚёкіј кҙҖкі„ л¶Ҳк°Җ
