EP 05 - format мқёмһҗмҷҖ лӢӨм–‘н•ң Renderer

rest_framework.response.Responseм—җм„ңлҠ” 2к°Җм§Җ нғҖмһ…мқҳ мқ‘лӢөмқ„ н•  мҲҳ мһҲмҠөлӢҲлӢӨ. (response.pyм—җм„ң ResponseнҒҙлһҳмҠӨ)

APIViewлҘј м“ёл•ҢлҠ” Response()мқҳ мқёмһҗлЎң мқ‘лӢөмқ„ н•ҳкІҢлҗҳлҠ”лҚ° ResponseлҠ” RendererмҶҚм„ұмқ„ м°ёмЎ°н•Ё >> renderer кё°ліёмңјлЎң 2к°Җм§ҖнҸ¬л§·мқ„л§Ң нҷңм„ұнҷ”лҗҳм–ҙмһҲлӢӨ.

  • api : API Endpointм—җ лёҢлқјмҡ°м ҖлҘј нҶөн•ҙ м ‘к·јн•  л•Ң, мӣ№UIлЎң APIлҘј мЎ°нҡҢн•  мҲҳ мһҲмҠөлӢҲлӢӨ.
  • json : ліҙнҶөмқҳ API м ‘к·ј мӢң

API мҡ”мІӯм—җм„ңлҠ” format=jsonмңјлЎң л””нҸҙнҠё мІҳлҰ¬лҗҳл©°, мӣ№лёҢлқјмҡ°м ҖлҘј нҶөн•ҙ мҡ”мІӯлҗ  л•Ңм—җлҠ” format=apiлЎң л””нҸҙнҠё мІҳлҰ¬лҗ©лӢҲлӢӨ.

APIViewм—җм„ңлҠ” м¶ңл ҘнҸ¬л§·мқ„ кІ°м •н•ҳлҠ” formatмқёмһҗлҘј лӢӨмқҢ 3к°Җм§Җ нҳ•нғңлЎң л°ӣмқ„ мҲҳ мһҲмҠөлӢҲлӢӨ.

  • Accept н—ӨлҚ”
  • GETмқёмһҗ format
  • URLлҒқм—җ вҖң.нҸ¬л§·вҖқ

м§Җм • мҳҲ) JSON мқ‘лӢө мҡ”мІӯ

м§Җм • мҳҲ) HTML мқ‘лӢө мҡ”мІӯ>>мқҙ нҳ•нғңк°Җ лёҢлқјмҡ°м ҖлЎңл“Өм–ҙк°Ҳл•Ңмқҳ rest-frameworkк°Җ ліҙм—¬мЈјлҠ”api

н—ӨлҚ”мҷҖ GETмқёмһҗлҘј нҷңмҡ©н•ҳлҠ” л°©лІ•мқҖ лі„лҸ„мқҳ URL Conf м„Өм •мқҙ н•„мҡ”м—ҶмҠөлӢҲлӢӨ. н•ҳм§Җл§Ң :8000/.jsonмқҳ кІҪмҡ°м—җлҠ” вҖңмқёмһҗ CaptureвҖқк°Җ н•„мҡ”н•ҳлҜҖлЎң лі„лҸ„мқҳ URL Conf м„Өм •мқҙ н•„мҡ”н•ҳл©° л·°м—җм„ңлҸ„ format мқҙлҰ„мқҳ мқёмһҗлҘј л°ӣмқ„ мҲҳ мһҲм–ҙм•ј н•©лӢҲлӢӨ.

Tip: мқёмһҗ captureлһҖ urlм—җм„ң <int:pk>/edit к°ҷмқҙ мҲ«мһҗмқёмһҗлҘј captureн•ҳлҠ”кІғмІҳлҹј м¶”м¶ңн•ҳлҠ”кІғ

URLConfм—җм„ң formatмқёмһҗм—җ лҢҖн•ң нҢЁн„ҙмқ„ м§Ғм ‘ м •мқҳн•  н•„мҡ”лҠ” м—ҶмҠөлӢҲлӢӨ. кё°мЎҙ URL Confм„Өм •мқ„ нҷңмҡ©н•ҳм—¬ format мқёмһҗ м§Җмӣҗмқ„ rest_frameworkм—җм„ң лҸ„мҷҖмӨҚлӢҲлӢӨ. л§Ңм•Ҫ лӢӨмқҢ 2к°Җм§Җ URLмқҙ мһҲлӢӨл©ҙ (routerм“°м§Җм•Ҡкі )

  • /post/
  • /post/1/

лӢӨмқҢ URLлҸ„ мӮ¬мҡ©нҶ лЎқ лҸ„мҷҖмӨҚлӢҲлӢӨ.

  • /post.json : нҸ¬мҠӨнҢ… лӘ©лЎқ (JSON нҸ¬л§·)
  • /post/1.json : 1лІҲ нҸ¬мҠӨнҢ… (JSON нҸ¬л§·)
  • /post.api : нҸ¬мҠӨнҢ… лӘ©лЎқ (HTML нҸ¬л§·)
  • /post/1.api : 1лІҲ нҸ¬мҠӨнҢ… (HTML нҸ¬л§·)

мӢӨм ң мҪ”л“ңлҘј мӮҙнҺҙлҙ…мӢңлӢӨ.

н•ҳм§Җл§Ң, DefaultRouterлҘј мӮ¬мҡ©н•ҳкі  кі„мӢңлӢӨл©ҙ, мң„ format_suffix_patternsмІҳлҰ¬лҘј м¶”к°ҖлЎң н•ҳмӢӨ н•„мҡ”к°Җ м—ҶмҠөлӢҲлӢӨ. DefaultRouterлӮҙм—җм„ң мқҙлҜё мІҳлҰ¬лҗҳм–ҙмһҲмҠөлӢҲлӢӨ.

ViewSetм—җм„ңлҠ” лі„лҸ„лЎң мӢ кІҪмҚЁмӨ„ кІғмқҖ м—Ҷмңјл©°, @api_viewмһҘмӢқмһҗлҘј м Ғмҡ©н•ң н•ЁмҲҳ л·°м—җм„ңлҠ” formatмқёмһҗлҘј л°ӣмқ„ мҲҳ мһҲлҸ„лЎқ Keyword мқёмһҗлҘј н•„нһҲ м§Җм •н•ҙмӨҳм•јн•©лӢҲлӢӨ. мқёмһҗл§Ң л°ӣмқ„ мҲҳ мһҲлҸ„лЎқ н•  лҝҗ м¶”к°ҖлЎң кө¬нҳ„н•ҙм•јн•  кІғмқҖ м—ҶмҠөлӢҲлӢӨ. api_viewмҷҖ ResponseлҘј нҶөн•ҙ formatмқёмһҗк°Җ мІҳлҰ¬лҗ©лӢҲлӢӨ.

Tip: urlм—җм„ң captureн•ҙм„ң keyword agrumentлЎң formatмқҙлқјлҠ” мқёмһҗлЎң л·°лЎң л„ҳкІЁмЈјлҠ”лҚ°, н•ЁмҲҳл·°м—җлҠ” formatмқ„ л°ӣмқ„мҲҳмһҲм–ҙм•јн•ҳлҜҖлЎң format=noneн•„мҡ”

мқҙм ң, лӢӨмқҢ URLмқ„ нҶөн•ҙ formatмқёмһҗлҘј м§Җм •н•ҙліҙм„ёмҡ”.

  • http :8000/ Accept:application/json
  • http :8000/?format=json
  • http :8000/.json
  • http :8000/ Accept:text/html
  • http :8000/?format=api
  • http :8000/.api

Tip:

мҰү лӮҙк°Җ м–ҙл–Ө мғҒнғңм—җм„ң м–ҙл–Ө мҡ”мІӯмқ„ н•ҳлғҗм—җл”°лқј лӢӨм–‘н•ң rendererк°Җ мӮ¬мҡ©лҗЁ

кё°ліё м§ҖмӣҗлҗҳлҠ” Renderer лӘҮ к°Җм§Җ(DRFм—җм„ң)

  • JSONRenderer (л””нҸҙнҠё м§Җм •) : json.dumpsлҘј нҶөн•ң JSON м§Ғл ¬нҷ”
    • format: json
  • BrowsableAPIRenderer (л””нҸҙнҠё м§Җм •) : self-document HTML л ҢлҚ”л§Ғ
    • format: api
  • TemplateHTMLRenderer : м§Җм • н…ңн”ҢлҰҝмқ„ нҶөн•ң л ҢлҚ”л§Ғ (apiмқёлҚ° мҷң htmlмқ„л¶ҖлҘҙлғҗ,,м°ЁлқјлҰ¬ djangoм“°мһҗ)
    • format: html
    • template_name: м§Җм • н•„мҡ”
    • н…ңн”ҢлҰҝмқ„ м°ҫлҠ” мҲңм„ң
      • Responseм—җм„ң template_name м°ҫкё° : return Response({}, template_name='mytempalte.html')
      • Rendererм—җм„ң template_name м°ҫкё°
      • л·° нҒҙлһҳмҠӨм—җм„ң get_template_names н•ЁмҲҳ м°ҫкё°
      • л·° нҒҙлһҳмҠӨм—җм„ң template_name мҶҚм„ұ м°ҫкё°

Tip: jsonл§ҢмңјлЎң renderer мқ‘лӢөн•ҙмЈјкі мӢ¶лӢӨл©ҙ мһ¬м •мқҳлҘј мқҙмҡ©н•ҳм—¬ BrewsableAPIRendererлҘј ләҙмЈјл©ҙлҗЁ

APIлҘј мӮ¬мҡ©н•ҳлҠ” HTML нҺҳмқҙм§Җ л§Ңл“Өкё°

м°ёкі : http://www.django-rest-framework.org/topics/html-and-forms/

Django Formмқ„ мӮ¬мҡ©н•ҳм§Җ м•Ҡкі , DRFлҘј нҶөн•ҙ HTMLнҺҳмқҙм§Җм—җм„ң; нҸјмІҳлҰ¬лҘј н•  мҲҳлҠ” мһҲмҠөлӢҲлӢӨ. н•ҳм§Җл§Ң мқҙ л¶Җл¶„мқҖ DRFлҘј м“°кё°ліҙлӢӨ Django Formмқ„ мӮ¬мҡ©н•ҳлҠ” кІғмқҙ ліҙлӢӨ нҡЁмңЁм Ғмқё л“Ҝ н•ҙм„ң, лӢӨлЈЁм§„ м•ҠкІ мҠөлӢҲлӢӨ.

м–ём  к°Җ Django Formкіј Serializerк°Җ н•©міҗм§ҖлҠ” лӮ мқҙ мҳӨм§Җ м•Ҡмқ„к№Ң мӢ¶л„Өмҡ”.

EP 06 - н•„н„°л§Ғ

лӘ©лЎқмЎ°нҡҢ APIView(ListAPIView)м—җм„ңлҠ” мЎ°кұҙм—җ л”°лқј QuerySetмқ„ н•„н„°л§Ғмқҙ н•„мҡ”н•  мҲҳ мһҲмҠөлӢҲлӢӨ.(нҠ№м • мң м Җм—җкІҢ ліҙм—¬мӨ„кІғмқҙ лӢӨлҘјл•Ң л“ұл“ұ) APIViewлҠ” ViewлҘј мғҒмҶҚл°ӣмқҖ CBVмқҙлҜҖлЎң, CBVм—җм„ң н•ҳлҚҳ лҢҖлЎң н•„н„°л§Ғмқ„ мҲҳн–үн•  мҲҳ мһҲмҠөлӢҲлӢӨ.

н•„н„°л§Ғм—җ н•„мҡ”н•ң мқёмһҗ нҡҚл“қ

self.requestлҘј нҶөн•ҙ HttpRequestк°қмІҙлҘј м°ёмЎ°н•  мҲҳ мһҲмҠөлӢҲлӢӨ.

  • self.request.user : нҳ„мһ¬ лЎңк·ёмқё мң м Җ Instance. лЎңк·ём•„мӣғ мӢңм—җлҠ” AnonymousUser(лӘЁлҚёнҒҙлһҳмҠӨк°ҖмһҲлҠ” мқјл°ҳ нҢҢмқҙмҚ¬ нҒҙлһҳмҠӨмқҙлҜҖлЎң мң м Җ лӘЁлҚё мқёмҠӨн„ҙмҠӨм—җ н•„н„°лӘ»н•ҙ, мқјл°ҳ нҢҢмқҙмҚ¬ нҒҙлһҳмҠӨмқҙлҜҖлЎң ???) мқёмҠӨн„ҙмҠӨ
  • self.request.GET : мҡ”мІӯ GETмқёмһҗ (QueryDictнғҖмһ…)
  • self.request.query_params : GETмқёмһҗмҷҖ лҸҷмқјн•ң к°’мһ…лӢҲлӢӨ. rest_frameworkм—җм„ңлҠ” ліҙлӢӨ к°ҖлҸ…м„ұлҶ’мқҖ мқҙлҰ„мңјлЎңм„ң мқҙ мҶҚм„ұмқ„ м§Җмӣҗн•ҳкі  мһҲмҠөлӢҲлӢӨ.

к·ёлҰ¬кі ,FBVм—җм„  urlм—җм„ң л„ҳм–ҙмҳӨлҠ” к°’мқ„ нҳ„мһ¬ usernameмңјлЎң мқёмһҗлҘј л°ӣкі  м§ҖкёҲмқҖ CBVмқҙкё°м—җ self.kwargsлҘј нҶөн•ҙ URL Captureлҗң мқёмһҗлҘј нҡҚл“қн•  мҲҳ мһҲмҠөлӢҲлӢӨ.(мӮ¬м „нҳ•мӢқмңјлЎң лӘЁл‘җ л„ҳм–ҙмҳҙ,) м•„лһҳмҷҖ к°ҷмқҙ URLнҢЁн„ҙмқҙ м •мқҳлҗҳм–ҙмһҲмқ„ кІҪмҡ°, self.kwargs['username']мңјлЎң н•ҙлӢ№ к°’мқ„ м°ёмЎ°н•  мҲҳ мһҲмҠөлӢҲлӢӨ.

foreignkeyм—җ лҢҖн•ҙм„ңлҠ” djangoм—җм„ң нҳ„мһ¬ On_delete лҘј л°ҳл“ңмӢң м§Җм •н•ҙм•јн•Ё

get_querysetмқ„ нҶөн•ң мҝјлҰ¬м…Ӣ н•„н„°л§Ғ

ListAPIViewм—җлҠ” querysetмқёмһҗлҘј м§Җм •н•ҳкі , мғҒнҷ©м—җ л”°лқј н•„н„°л§Ғмқҙ н•„мҡ”н•ң кІҪмҡ°м—җлҠ” get_querysetн•ЁмҲҳлҘј мһ¬м •мқҳн•ҙм„ң кө¬нҳ„н•  мҲҳ мһҲмҠөлӢҲлӢӨ.

Django Adminмқҳ Searchкё°лҠҘ(мҰү, мһҘкі  admin.pyм—җм„ң search_fields = (вҖҳtitleвҖҷ, вҖҳcontentвҖҷ)м¶”к°Җн•ҳл©ҙ adminнҷҳкІҪм—җм„ң кІҖмғүкё°лҠҘ нҷңм„ұнҷ”лҗҳлҠ”кІғмІҳлҹј..)кіј мң мӮ¬н•ҳкІҢ м ңкіөн•©лӢҲлӢӨ. мқҙлҠ” лі„лҸ„мқҳ кІҖмғүм—”м§„мқ„ мӮ¬мҡ©н•ҳлҠ” кІғмқҙ м•„лӢҲлқј, DBMSмқҳ LIKE/ILIKE мЎ°кұҙм Ҳмқ„ нҷңмҡ©н•©лӢҲлӢӨ. EP 07. Pagination мІҳлҰ¬(ep03м•ұмқ„ мқҙмҡ©)

л ҲмҪ”л“ң к°ҜмҲҳк°Җ л§ҺмқҖ кІҪмҡ° лӘ©лЎқмқ„ н•ҳлӮҳмқҳ API мҡ”мІӯл§ҢмңјлЎң л°ӣлҠ” кІғмқҖ н”јн•ҙм•јн•  кІғмһ…лӢҲлӢӨ. мқҙлҹҙ л•Ң м—¬лҹ¬ нҺҳмқҙм§Җм—җ лӮҳлҲ м„ң мҡ”мІӯмқ„ мІҳлҰ¬н•  мҲҳ мһҲкІ лҠ” лҚ°мҡ”. мқҙм—җ лҢҖн•ҙ rest_frameworkм—җм„ңлҠ” м—¬лҹ¬ нҺҳмқҙм§• кё°лІ•мқ„ м§Җмӣҗн•ҙмЈјкі  мһҲмҠөлӢҲлӢӨ.

  • PageNumberPagination:pageмқёмһҗлҘј нҶөн•ҙ нҺҳмқҙм§• мІҳлҰ¬ (pageлҠ” нҺҳмқҙм§ҖмҲҳ, page_sizeлҠ” pageлӢ№ postмҲҳ)
    • http://api.example.org/accounts/?page=4
    • http://api.example.org/accounts/?page=4&page_size=100
  • LimitOffsetPagination:limitмқёмһҗлҘј нҶөн•ң нҺҳмқҙм§• мІҳлҰ¬(offsetмқҖ л ҲмҪ”л“ңм •л ¬мҲңм„ңкё°мӨҖ 400лІҲм§ёл¶Җн„°~, limit к№Ңм§Җ, limitл§Ң м§Җм •мӢң мІҳмқҢл¶Җн„° limitк№Ңм§Җ)
    • http://api.example.org/accounts/?limit=100
    • http://api.example.org/accounts/?offset=400&limit=100

rest_framework/generics.py лӮҙ GenericAPIViewм—җлҠ” мқҙлҜё pagination_class = PageNumberPagination м„Өм •мқҙм—ҲмҠөлӢҲлӢӨ. н•ҳм§Җл§Ң, 3.7.0лІ„м „л¶Җн„°лҠ” л””нҸҙнҠё NoneмңјлЎң м§Җм •мңјлЎң ліҖкІҪлҗҳм—ҲмҠөлӢҲлӢӨ. нҳ„мһ¬ (2017л…„ 11мӣ”) мөңмӢ  лІ„м „мқҖ 3.7.3лІ„м „мһ…лӢҲлӢӨ.
