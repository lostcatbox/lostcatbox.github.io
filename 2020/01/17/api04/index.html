EP 04 - ViewSetкіј Router

ViewSet

ViewSetмқҖ мқјл°ҳм Ғмқё мһҘкі  CBVлҠ” м•„лӢҷлӢҲлӢӨ. 2к°ңмқҳ л·°лҘј л§Ңл“Өм–ҙмЈјлҠ” ліҙлӢӨ нҷ•мһҘлҗң нҳ•нғңмқҳ CBVмһ…лӢҲлӢӨ. вҖңView + SetвҖқ мҰү, лӢӨмҲҳ ViewлҘј м§Җмӣҗн•ҳлҠ” CBV.

лӘЁл“  л·°м…ӢмқҖ .as_view({'http_method': 'мІҳлҰ¬н• л©ӨлІ„н•ЁмҲҳ'})лҘј нҳём¶ңн•ҳм—¬, н•ҙлӢ№ http_methodлҘј м§Җмӣҗн•ҳлҠ” л·° н•ЁмҲҳлҘј мғқм„ұн•©лӢҲлӢӨ. 1к°ңмқҳ л·° н•ЁмҲҳлҘј мғқм„ұн•ҳлҜҖлЎң, н•ҳлӮҳмқҳ URLл§Ңмқ„ мІҳлҰ¬н•  мҲҳ мһҲмҠөлӢҲлӢӨ.

rest_framework/viewsets.pyм—җм„ңлҠ” лӢӨмқҢ 2к°Җм§Җ л·°м…Ӣмқ„ м§Җмӣҗн•ҳкі  мһҲмҠөлӢҲлӢӨ.

: лӘ©лЎқ мЎ°нҡҢ, нҠ№м • л ҲмҪ”л“ң мЎ°нҡҢлҘј м§Җмӣҗ => 2к°ңмқҳ URL м§Җмӣҗ

  • нҠ№м • Record мЎ°нҡҢ : mixins.RetrieveModelMixinмқ„ нҶөн•ҙ retrieve()н•ЁмҲҳ м§Җмӣҗ
  • лҰ¬мҠӨнҠё мЎ°нҡҢ : mixins.ListModelMixinмқ„ нҶөн•ҙ list()н•ЁмҲҳ м§Җмӣҗ

: лӘ©лЎқ мЎ°нҡҢ, мғқм„ұ, нҠ№м • л ҲмҪ”л“ң мЎ°нҡҢ/мҲҳм •/мӮӯм ң м§Җмӣҗ => 2к°ңмқҳ URL м§Җмӣҗ

  • лҰ¬мҠӨнҠё мЎ°нҡҢ : mixins.ListModelMixinмқ„ нҶөн•ҙ list()н•ЁмҲҳ м§Җмӣҗ
  • нҠ№м • Record мЎ°нҡҢ : mixins.RetrieveModelMixinмқ„ нҶөн•ҙ retrieve()н•ЁмҲҳ м§Җмӣҗ
  • мғҲ Record мғқм„ұ : mixins.CreateModelMixinмқ„ нҶөн•ҙ create()н•ЁмҲҳ м§Җмӣҗ
  • нҠ№м • Record мҲҳм • : mixins.UpdateModelMixinмқ„ нҶөн•ҙ update()н•ЁмҲҳ л°Ҹ partial_update()н•ЁмҲҳ м§Җмӣҗ
  • нҠ№м • Record мӮӯм ң : mixins.DestroyModelMixinмқ„ нҶөн•ҙ destroy()н•ЁмҲҳ м§Җмӣҗ

лӢӨмқҢ мҪ”л“ңм—җм„ң ліҙл“Ҝмқҙ, PostViewSetм—җм„ң list/create/retrieve/update/partial_update/destory н•ЁмҲҳлҘј лӘЁл‘җ м§Җмӣҗн•ңлӢӨкі  н•ҳм—¬, мқҙ н•ЁмҲҳл“Өмқ„ н•ҳлӮҳмқҳ URLм—җм„ң мІҳлҰ¬н•  мҲҳ м—ҶмҠөлӢҲлӢӨ. мқјл°ҳм Ғмқё REST APIм„Өкі„м—җм„ң лІ—м–ҙлӮҳкё° л•Ңл¬ёмһ…лӢҲлӢӨ. >>кІ°көӯ url 2к°ң м§Җмӣҗн•ҙм•јн•Ё

REST API к·ңкІ©м—җ л§һм¶°, URL л§Өн•‘мқ„ н•ҙлҙ…мӢңлӢӨ.

list route(Pkк°’ н•„мҡ”м—ҶмқҢ)

detail route(Pkк°’ н•„мҡ”н•Ё)

мқҙл ҮкІҢ л§Ңл“Өм–ҙм§„ л·° н•ЁмҲҳлҠ” лӢӨлҘё FBV(н•ЁмҲҳ кё°л°ҳ л·°)мҷҖ лҸҷмқјн•ҳкІҢ URLConfм—җ л§Өн•‘н•  мҲҳ мһҲмҠөлӢҲлӢӨ.

мқҪкё° м „мҡ© л·°к°Җ н•„мҡ”н•  кІҪмҡ°, лӢӨмқҢкіј к°ҷмқҙ л§Өн•‘н•ҳмӢӨ мҲҳ мһҲмҠөлӢҲлӢӨ.

Router нҷңмҡ©

RouterлҘј нҷңмҡ©н•ҳл©ҙ, кҙҖлЎҖ(Convention)м—җ кё°л°ҳн•ҳм—¬ URLл§Өн•‘мқ„ мҶҗмүҪкІҢ н•  мҲҳ мһҲмҠөлӢҲлӢӨ. м•„лһҳ мҪ”л“ңм—җм„ң ліҙл“Ҝмқҙ PostViewSetмқ„ Routerм—җ л“ұлЎқн•ҳл©ҙ, н•ҙлӢ№ л·°м…Ӣмқҙ м§Җмӣҗн•ҳлҠ” л©”мҶҢл“ң/н•ЁмҲҳл“Өм—җ н•ңн•ҙм„ң URLл§Өн•‘мқ„ мҲҳн–үн•©лӢҲлӢӨ. л·°лҠ” л“ұлЎқн•  мҲҳ м—ҶмҠөлӢҲлӢӨ. л·°м…Ӣл§Ң л“ұлЎқк°ҖлҠҘн•©лӢҲлӢӨ.

н•ҳлӮҳмқҳ Routerм—җ лӢӨмҲҳ л·°м…Ӣмқ„ л“ұлЎқн•ҳмӢӨ мҲҳ мһҲмҠөлӢҲлӢӨ.

л””нҸҙнҠё л§Өн•‘мқҖ мң„м—җм„ң мҲҳн–үн•ң лӮҙм—ӯлҢҖлЎң лҰ¬мҠӨнҠё/л””н…Ңмқј лқјмҡ°нҢ…мқ„ мҲҳн–үн•©лӢҲлӢӨ.

  • list routeм—җм„ңлҠ” /prefix/мЈјмҶҢк°Җ м§Җм •лҗҳл©°, URL Reverse мқҙлҰ„мңјлЎңм„ң лӘЁлҚёлӘ…мҶҢл¬ёмһҗ-list

    к°Җ мӮ¬мҡ©лҗ©лӢҲлӢӨ.

    • get => list
    • post => create

  • detail routeм—җм„ңлҠ” /prefix/pk/ мЈјмҶҢк°Җ м§Җм •лҗҳл©°, URL Reverse мқҙлҰ„мңјлЎңм„ң лӘЁлҚёлӘ…мҶҢл¬ёмһҗ-detail мқҙ мӮ¬мҡ©лҗ©лӢҲлӢӨ.

    • get => retrieve
    • put => update
    • patch => partial_update
    • delete => destroy

мҳҲ1)

лӢӨмқҢкіј к°ҷмқҙ registerлҗ  кІҪмҡ°

  • /post/ мЈјмҶҢм—җ лҢҖн•ҙ URL Reverse мқҙлҰ„мқҖ post-listмқҙ л“ұлЎқлҗ©лӢҲлӢӨ.(name=post-listмһ„)
  • /post/10/ лҘҳмқҳ мЈјмҶҢм—җ лҢҖн•ҙ URL Reverse мқҙлҰ„мқҖ post-detailмқҙ л“ұлЎқлҗ©лӢҲлӢӨ.

Tip: м•„лһҳлҠ” rest-frameworkм—җ routers.py лӮҙмҡ©мӨ‘ н•ҳлӮҳмқҙлӢӨ, ліҙл©ҙquerysetм—җм„ң modelк°Җм ёмҷҖм„ң _metaлҚ°мқҙн„°м—җм„ң мқҙлҰ„к°Җм ёмҷҖм„ң мҶҢл¬ёмһҗлЎң л°ӣлҠ”кІғмқ„ м•ҢмҲҳмһҲлӢӨ. basenameмқҖ кІ°көӯ мң„м—җ reverse nameмқё post-list, post-detailм—җм„ң postк°Җ лҚ°мқҙн„°мқҙлҰ„м—җ л”°лқј л°”лҖ”мҲҳмһҲлӢӨлҠ”кІғмһ„

мҳҲ2)

лӢӨмқҢкіј к°ҷмқҙ registerлҗ  кІҪмҡ°

  • /hello/ мЈјмҶҢм—җ лҢҖн•ҙ URL Reverse мқҙлҰ„мқҖ post-listк°Җ л“ұлЎқлҗ©лӢҲлӢӨ.
  • /hello/10/ лҘҳмқҳ мЈјмҶҢм—җ лҢҖн•ҙ URL Reverse мқҙлҰ„мқҖ post-detailмқҙ л“ұлЎқлҗ©лӢҲлӢӨ.

routerлҘј м“ё кІҪмҡ°лҠ” api-rootл·°лҘј нҶөн•ң м§Җмӣҗ ViewSet лӘ©лЎқ мЎ°нҡҢ

Routerм—җм„ңлҠ” м¶”к°ҖлЎң нҳ„ Routerм—җ л“ұлЎқлҗң ViewSetлӮҙм—ӯмқ„ мЎ°нҡҢн•  мҲҳ мһҲлҠ” api-root л·°лҘј м¶”к°ҖлЎң м§Җмӣҗн•©лӢҲлӢӨ. router.urlsк°Җ л§Өн•‘лҗң мЈјмҶҢлЎң лёҢлқјмҡ°м Җ м ‘мҶҚ нҳ№мқҖ GETмҡ”мІӯмқ„ лӮ л Өліҙм„ёмҡ”. ;)

ViewSetм—җ м¶”к°Җ APIлҠ” м–ҙл–»кІҢ м¶”к°Җн•  .. мҲҳ мһҲмЈ  ??

л¬јлЎ мһ…лӢҲлӢӨ. list routeлЎңм„ң list/createн•ЁмҲҳмҷҖ detail routeлЎңм„ң retrieve/update/partial_update/deleteмҷём—җ м¶”к°ҖлЎң кө¬нҳ„н•ҳм—¬ л§Өн•‘н•ҳмӢӨ мҲҳ мһҲмҠөлӢҲлӢӨ. мқҙл•Ң м¶”к°Җн•  APIк°Җ list routeм—җ л“ұлЎқн•  кІғмқём§Җ, detail routeм—җ л“ұлЎқн•  кІғмқём§ҖлҘј кІ°м •н•ҙм•јн•©лӢҲлӢӨ.(pkк°’н•„мҡ”мң л¬ҙм°Ёмқҙ)

кө¬нҳ„мқҖ н•ҙлӢ№ ViewSet нҒҙлһҳмҠӨ лӮҙм—җ л©ӨлІ„н•ЁмҲҳлЎңм„ң кө¬нҳ„н•ҳкі  list_route мһҘмӢқмһҗ нҳ№мқҖ detail_route мһҘмӢқмһҗлЎң кҫёл©°мЈјл©ҙ лҒ„мқ• !!! URLл§Өн•‘мқҖ Routerм—җм„ң м•Ңм•„м„ң н•ҙмӨҚлӢҲлӢӨ.

Django-rest-framework 3.8лІ„м „мқҙмғҒл¶Җн„°лҠ” actionмңјлЎң нҶөн•©лҗЁ

PostViewSetм—җ лӢӨмқҢ 2к°ң APIлҘј м¶”к°Җн•ҙлҙ…мӢңлӢӨ. м•„лһҳ мҳҲмӢңлҠ” PostлӘЁлҚём—җ is_public=models.BooleanField()к°Җ мһҲмқҢмқ„ к°Җм •н•ң мҪ”л“ңмһ…лӢҲлӢӨ.

PostViewSetмқҳ public_list API

  • URL л§Өн•‘мқҖ /prefix/н•ЁмҲҳлӘ…/мңјлЎңм„ң /post/public_list/к°Җ лҗ©лӢҲлӢӨ.
  • methodsк°Җ м§Җм •лҗҳм§Җ м•Ҡм•ҳмңјлҜҖлЎң GETмҡ”мІӯм—җ мқ‘лӢөн•©лӢҲлӢӨ. мқҙлҠ” list_route/detail_routeмқҳ л””нҸҙнҠё мІҳлҰ¬мһ…лӢҲлӢӨ.
  • н•ЁмҲҳлӘ…мқҙ public_listмқҙлҜҖлЎң, URL Reverse мқҙлҰ„мқҖ лӘЁлҚёлӘ…-н•ЁмҲҳлӘ…мңјлЎңм„ң post-public-listк°Җ лҗ©лӢҲлӢӨ. м–ёлҚ”л°”(_)лҠ” н•ҳмқҙн”Ҳ(-)мңјлЎң ліҖкІҪн•ҳм—¬ мІҳлҰ¬лҗ©лӢҲлӢӨ.

мҡ”мІӯ мҳҲмӢң

PostViewSetмқҳ set_public API

  • URL л§Өн•‘мқҖ /prefix/{pk}/н•ЁмҲҳлӘ…/мңјлЎңм„ң /post/{pk}/set_public/мқҙ лҗ©лӢҲлӢӨ.
  • methodsмқҙ patchлЎң м§Җм •лҗҳм—ҲмңјлҜҖлЎң, patch мҡ”мІӯм—җ мқ‘лӢөн•©лӢҲлӢӨ.
  • н•ЁмҲҳлӘ…мқҙ set_publicмқҙлҜҖлЎң, URL Reverse мқҙлҰ„мқҖ лӘЁлҚёлӘ…-н•ЁмҲҳлӘ…мңјлЎңм„ң post-set-publicмқҙ лҗ©лӢҲлӢӨ. м–ёлҚ”л°”(_)лҠ” н•ҳмқҙн”Ҳ(-)мңјлЎң ліҖкІҪн•ҳм—¬ мІҳлҰ¬лҗ©лӢҲлӢӨ.

мҡ”мІӯ мҳҲмӢң

ліё APIлҠ” detail_routeмҳҲлЎңм„ң мһ‘м„ұн•ң кІғмқј лҝҗ, лӢӨмқҢкіј к°ҷмқҙ PATCHмҡ”мІӯмқ„ нҶөн•ң partial_updateмқ„ мқҙмҡ©н•ҙлҸ„ м¶©л¶„н•©лӢҲлӢӨ.

Tip: PUTмқ„ нҶөн•ң мҲҳм •м—җм„ңлҠ” лӘЁл“  н•„мҲҳ(required) н•„л“ңлҘј м§Җм •н•ҳм—¬м•ј н•ҳл©°, PATCHлҘј нҶөн•ң мҲҳм •м—җм„ңлҠ” вҖңл¶Җл¶„мҲҳм •вҖқмңјлЎңм„ң мҲҳм •н•  н•„л“ңл§Ң м§Җм •н•ҳл©ҙ лҗ©лӢҲлӢӨ.
