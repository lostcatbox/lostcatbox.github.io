EP 08 - SerializerвЦ╝ ьєхьЋю ВюаьџеВё▒ Ж▓ђВѓг в░Ј ВађВъЦ

SerializerВЮў ВЃЮВё▒Въљ

Serializerвіћ Django FormЖ│╝ В╗еВЁЅ/ВѓгВџЕв▓ЋВЮ┤ ВюаВѓгьЋЕвІѕвІц. ьЋўВДђвДї ВЃЮВё▒ВъљвЦ╝ ВДђВаЋьЋа вЋї, ВЮИВъљ ЖхгВё▒ВЮ┤ ВА░ЖИѕ ВЃЂВЮ┤ьЋювЇ░Вџћ.

Django FormВЮў ВЃЮВё▒Въљ Signatureвіћ вІцВЮїЖ│╝ Ж░ЎВіхвІѕвІц. В▓Фв▓ѕВДИ ВЮИВъљвАю dataвЦ╝ в░ЏВіхвІѕвІц.

ModelFormВЌљВёювіћ instanceВЮИВъљЖ░ђ ВХћЖ░ђвАю ВДђВаЋвљўВќ┤ ВъѕЖхгВџћ.(instanceВЮИВъљвЦ╝ ВБ╝вЕ┤ ВДђВаЋ instance ВађВъЦВЮё ВѕўьќЅЖ░ђвіЦьЋўЖ▓ївѓ┤вХђВаЂВю╝вАю ВёцВаЋвље)

ЖиИвъўВёю, ви░ вѓ┤ВЌљВёю FormВЃЮВё▒ВъљвЦ╝ ьўИВХюьЋа вЋїВЌљвіћ вІцВЮїЖ│╝ Ж░ЎВЮђ ВйћвЊюЖ░ђ Ж░ђвіЦьЋЕвІѕвІц.

вДјВЮ┤ ВЮхВѕЎьЋўВІа ВйћвЊюВБа? :)

ЖиИвЪ░вЇ░, SerializerВЮў ВЃЮВё▒Въљ Signatureвіћ вІцВЮїЖ│╝ Ж░ЎВіхвІѕвІц. В▓Фв▓ѕВДИ ВЮИВъљвАю instanceвЦ╝ в░ЏВю╝вЕ░, вЉљв▓ѕВДИ ВЮИВъљвАю dataвЦ╝ в░ЏВіхвІѕвІц.

Django FormВЮў SignatureЖ░ђ ВА░ЖИѕ вІцвдЁвІѕвІц. dataВЮИВъљвДї ВДђВаЋьЋўВІц вЋїВЌљвіћ MySerializer(data=mydata)В▓ўвЪ╝ ьЋёьъѕ keywordвЦ╝ ВДђВаЋьЋ┤ВБ╝ВЁћВЋ╝ ьЋЕвІѕвІц.

В░ИЖ│а) ьїїВЮ┤ВЇгВЮў ьЋеВѕў ВЮИВъљВЌљ вїђьЋ┤Вёювіћ РђюьїїВЮ┤ВЇг В░еЖи╝В░еЖи╝ ВІюВъЉьЋўЖИ░ - ьЋеВѕўьјИРђю ВЌљВёю ВъљВёИьъѕ вІцвБеЖ│а ВъѕВіхвІѕвІц.

data= ВЮИВъљЖ░ђ ВБ╝Вќ┤ВДђвЕ┤, вІцВЮї ВѕюВёювАю В▓ўвдгвљЕвІѕвІц.

  1. .is_valid()Ж░ђ ьўИВХюВЮ┤ вљўвЕ┤
  2. .initial_data ьЋёвЊюВЌљ ВаЉЖи╝ьЋа Вѕў ВъѕЖ│а,
  3. .validated_data (ВѓгВаёьўЋьЃю)вЦ╝ ьєхьЋ┤ ВюаьџеВё▒ Ж▓ђВдЮВЌљ ьєхЖ│╝ьЋю Ж░њвЊцВЌљ вїђьЋю ВѓгВаё. .save()ВІюВЌљ ВѓгВџЕвљЕвІѕвІц.(cleaned_dataЖ░ЎВЮђвіљвѓї)
  4. .errors : ВюаьџеВё▒ Ж▓ђВѓгВЌљ вїђьЋю ВўцвЦў вѓ┤ВЌГ
  5. .data : ВюаьџеВё▒ Ж▓ђВѓг ьЏёВЌљ, Ж░▒ВІављю ВЮИВіцьё┤ВіцВЌљ вїђьЋю ьЋёвЊюЖ░њ ВѓгВаё(restframeworkВЌљВёю ьі╣ВаЋ ВЮИВіцьё┤ВіцВЮў ьЋёвЊювѓ┤ВЌГВЮё ВЮЉвІхьЋавќё ВѓгВџЕ)

serializer.save(**kwargs) ьўИВХюВЮ┤ ВЮ┤вцёВДѕ вЋї (form.saveвіљвѓї)

  • DBВЌљ ВађВъЦвљю Ж┤ђвае вфевЇИ ВЮИВіцьё┤ВіцвЦ╝ вдгьё┤
  • .validated_dataВЎђ kwargsВѓгВаёВЮё ьЋЕВ│љВёю, .update ьЋеВѕў/.createьЋеВѕўвЦ╝ ьєхьЋ┤ Ж┤ђвае ьЋёвЊюВЌљ Ж░њВЮё ьЋавІ╣ьЋўЖ│а, DBвАюВЮў ВађВъЦВЮё ВІювЈёьЋЕвІѕвІц.
    • .update() : self.instance ВЮИВъљЖ░ђ ВДђВаЋвљўВЌѕВЮё вЋї
    • .create() : self.instance ВЮИВДђЖ░ђ ВДђВаЋвљўВДђ ВЋіВЋўВЮё вЋї

Validators

ВъЦЖ│а ЖИ░в│И validatorsЖ│╝ вЇћвХѕВќ┤, django-rest-frameworkВЌљВёювіћ ВюаВЮ╝Вё▒ ВЌгвХђ В▓┤ьЂгвЦ╝ вЈёВЎђВБ╝віћ ValidatorвЦ╝ ВаюЖ│хьЋ┤ВБ╝вЕ░, queryset в▓ћВюёвЦ╝ ВаюьЋюьЋўВЌг ВДђВаЋ в▓ћВюё вѓ┤ВЌљВёюВЮў ВюаВЮ╝Вё▒ ВЌгвХђвЦ╝ В▓┤ьЂгьЋа Вѕў ВъѕВіхвІѕвІц.

  1. UniqueValidator : ВДђВаЋ ьЋёвЊюЖ░ђ ВДђВаЋ QuerySetв▓ћВюёВЌљВёю UniqueьЋюВДђ В▓┤ьЂг
    • вфевЇИ ьЋёвЊю unique=True ВёцВаЋВЌљ вїђВЮЉьЋўВЌг, ВъљвЈЎ ВХћЖ░ђ
  2. UniqueTogetherValidator
    • вфевЇИ ьЂ┤въўВіц.Meta.unique_together ВєЇВё▒ВЌљ вїђВЮЉьЋўВЌг, ВъљвЈЎ ВХћЖ░ђ
  3. UniqueForDateValidator
    • вфевЇИ ьЋёвЊю unique_for_date=True ВёцВаЋВЌљ вїђВЮЉьЋўВЌг, ВъљвЈЎ ВХћЖ░ђ
  4. UniqueForMonthValidator
    • вфевЇИ ьЋёвЊю unique_for_month=True ВёцВаЋВЌљ вїђВЮЉьЋўВЌг, ВъљвЈЎ ВХћЖ░ђ
  5. UniqueForYearValidator
    • вфевЇИ ьЋёвЊю unique_for_year=True ВёцВаЋВЌљ вїђВЮЉьЋўВЌг, ВъљвЈЎ ВХћЖ░ђ

ВюаьџеВё▒ Ж▓ђВѓг ВўѕВЎИ

rest_framework.exceptions.ValidationErrorвЦ╝ ЖИ░в│ИВю╝вАю ВѓгВџЕьЋЕвІѕвІц. ВЮ┤віћ ВЮЉвІх ВЃЂьЃюВйћвЊю 400Вю╝вАю В▓ўвдгьЋЕвІѕвІц.

ВъЦЖ│а ЖИ░в│ИВЌљВёю ВаюЖ│хьЋўвіћ django.core.exceptions.ValidationErrorвЦ╝ ВѓгВџЕьЋа ВѕўвЈё ВъѕВіхвІѕвІц. ВЮ┤вЦ╝ ВѓгВџЕьЋўвЕ┤, rest_frameworkВИА ВўѕВЎИвАю в│ђьЎўвљўВќ┤ В▓ўвдгвљЕвІѕвІц.

SerializerВЌљВёю ВюаьџеВё▒Ж▓ђВѓг ьЋеВѕў ВДђВаЋьЋўЖИ░

Tip: ModelSerializerвЦ╝ ВѓгВџЕьЋўВІавІцвЕ┤, ВюаьџеВё▒ Ж▓ђВѓг ьЋеВѕўвіћ вфевЇИ ВИАВЌљ ВДђВаЋьЋўВІювіћ Ж▓ЃВЮ┤ Ж┤ђвдгВИАвЕ┤ВЌљВёю ВбІВіхвІѕвІц.

(ЖиИваЄЖ▓ї ьЋўвЕ┤ django.adminВЌљВёювЈё, apiВЌљВёювЈё, modelВъљВ▓┤ВЌљВёювЈё ВюаьџеВё▒Ж▓ђВѓгвљўв»ђвАю)

  • ьЋёвЊю ВаЋВЮў ВІюВЌљ validatorsВЮИВъљ ВДђВаЋьЋўЖИ░
  • Field-level Ж▓ђВѓг : ьі╣ВаЋ ьЋёвЊюВЌљ вїђьЋю Ж▓ђВѓг
    • FormВЌљВёювіћ clean_ьЋёвЊювфЁВЮў ьЋеВѕўвЦ╝ ЖхгьўёьЋўВДђвДї, rest_frameworkВЌљВёювіћ validate_ьЋёвЊювфЁвЦ╝ ЖхгьўёьЋ┤ВцЇвІѕвІц.
    • ьЋеВѕў ВЮИВъљвАю ьЋ┤вІ╣ Ж░њВЮ┤ ВаёвІгвљЕвІѕвІц.(value)
    • DjangoFormЖ│╝ вДѕВ░гЖ░ђВДђвАю, в│И ьЋеВѕўВЮў вдгьё┤Ж░њВЮё ьєхьЋ┤ Ж░њВЮё в│ђьЎўьЋа ВѕўвЈё ВъѕВіхвІѕвІц.

  • Object-level Ж▓ђВѓг : вІцВѕў ьЋёвЊюВЌљ вїђьЋю Ж▓ђВѓг

    (ВўѕВІювАю Ж▓їВъёВЌљВёю ВЋёВЮ┤вћћ ВцЉв│хВЮё Ж▓ђВѓгьЋавЋї Вќ┤вќц Вёюв▓ёВЌљВёю Вќ┤вќцВЮ┤вдё ВцЉв│хвЦ╝ В░ЙВЋёВЋ╝ьЋўв»ђвАю Вёюв▓ёВЎђ ВЮ┤вдёВЮё вЈЎВІюВЌљ ВюаьџеВё▒Ж▓ђВѓг)

DBвАюВЮў в░ўВўЂЖ│╝ MixinsВЮў perform_ Ж│ёВЌ┤ ьЋеВѕў

(ВўѕВІю:ModelViewSetвЈё Ж╣їв│┤вЕ┤ ВЋёвъўВЌљ mixinвЊц ВЮ┤ВџЕьЋе)

APIВѕўьќЅЖ▓░Ж│╝вЦ╝ DBВЌљ в░ўВўЂьЋўвіћ create/update/destroyвЦ╝ В╗цВіцьЁђьЋўЖ│а ВІХВю╝ВІювІцвЕ┤, perform_ Ж│ёВЌ┤ ьЋеВѕўВЌљ ВБ╝вфЕьЋўВёИВџћ.

  • ВађВъЦьЋа вЋї
    • ВЋёВЮ┤ьћ╝ ВађВъЦьЋўЖИ░
    • ВюаВађ ВаЋв│┤ ЖИ░вАЮьЋўЖИ░

createВІюВЌљ ВХћЖ░ђвАю ьЋёвЊю ВёцВаЋьЋўЖИ░

CREATEЖИ░віЦВЮё ВѕўьќЅьЋўвіћ APIViewВЌљВёювіћ rest_framework.mixins.CreateModelMixinВЮё ВЃЂВєЇв░ЏВіхвІѕвІц. CREATEВІюВЌљ ВХћЖ░ђвАю ьЋёвЊювЦ╝ ВёцВаЋьЋавацвЕ┤, perform_createвЦ╝ ьєхьЋ┤ ВѕўьќЅвљўвіћ serializer.save()ьЋеВѕўВЌљ ьѓцВЏївЊю ВЮИВъљвЦ╝ ВДђВаЋьЋ┤ВБ╝ВёИВџћ.

ьўёВъг ВўцвЦўЖ░ђ вѓўВўцвіћ ВЮ┤Вюа serializers.py ВЌљВёювіћ fieldsВЌљВёю ьі╣ВаЋ ьЋёвЊювДї ВДђВаЋьќѕВю╝вфевАю ВюаьџеВё▒ Ж▓ђВѓгвЦ╝ Въў ьєхЖ│╝ьЋ┤вЈё

вДѕВДђвДЅВЌљ serializer.save()Ж░ђ вЊцВќ┤Ж░ѕвЋї вІцвЦИ ьЋёВѕў fieldЖ░ђ ВДђВаЋвљўВќ┤ВъѕВДђ ВЋівІцвЕ┤ not null constraint failedЖ░ђ вю░Вѕўв░ќВЌљВЌєвІц.
