EP 09 - Authenticationкіј Permissions

мқҙм ң API мҡ”мІӯмқ„ мІҳлҰ¬н•  л•Ң м ‘к·јм ңн•ңмқ„ м§Җмӣҗн•ҙлҙ…мӢңлӢӨ. м•„л¬ҙлӮҳ лӢӨлҘё мӮ¬лһҢмқҳ кёҖмқ„ мҲҳм •/мӮӯм ңн•  мҲҳ мһҲм–ҙм„ңлҠ” м•Ҳ лҗҳкІ мҠөлӢҲлӢӨ.

Djangoм—җм„ңлҠ” authм•ұмқ„ нҶөн•ҙ UserлӘЁлҚёмқ„ м§Җмӣҗн•ҙмЈјкі  мһҲмҠөлӢҲлӢӨ.

UserлӘЁлҚёмқ„ нҶөн•ҙ PostлӘЁлҚём—җ мһ‘м„ұмһҗлҘј кё°лЎқн•ҙлҙ…мӢңлӢӨ. (нҳ„мһ¬ ep08м—җм„ңмһ‘м—…мӨ‘)

мҳӨлҘҳлҠ” мқҙлҜё ep04м—җм„ң ForeignKeyмқҙлҜё лҳ‘к°ҷмқҙ мӮ¬мҡ©мӨ‘мқҙлҜҖлЎң м—¬кё°м—җм„ң reverseлҘј н• л•Ң л‘ҳмӨ‘л¬ҙм—Үмқ„ лҸҢл ӨмӨҳм•јн• м§ҖкІ№міҗлІ„лҰ¬кІҢлҗңлӢӨ. л”°лқјм„ң related_name='ep08_post_set'лҘј м§Җм •н•ҙмЈјл©ҙ ep04м—җм„ңлҠ” user.post_set.all() мңјлЎң кҙҖл Ё нҸ¬мҠӨнҠём—җ лҢҖн•ң мҝјлҰ¬ лӘЁл‘җк°Җм ёмҳ¬мҲҳмһҲкі , ep08лӘЁлҚём—җм„ңлҠ” user.ep08_post_set.all() мңјлЎң кҙҖл Ё нҸ¬мҠӨнҠём—җ лҢҖн•ңмҝјлҰ¬ лӘЁл‘җ к°Җм ёмҳ¬мҲҳмһҲлӢӨ.

н•„мҲҳн•„л“ңлЎңм„ң author н•„л“ңлҘј м¶”к°Җн–Ҳкё° л•Ңл¬ём—җ makemigrations лӘ…л № мҲҳн–ү мӢңм—җ м§Ҳл¬ёмқҙ лӮҳмҳөлӢҲлӢӨ. л§Ҳмқҙк·ёл Ҳмқҙм…ҳм—җ лҢҖн•ҙм„ң к°Җл¬јк°Җл¬јн•ҳмӢ  л¶„мқҖ вҖңмһҘкі  кё°ліёнҺё MigrationвҖң VOD м—җн”јмҶҢл“ңлҘј м°ёкі н•ҳм„ёмҡ”.

мқҙм ң json api мҡ”мІӯмқ„ н•ҳл©ҙ, authorл¶Җл¶„мқҖ Primary Keyк°’мқҙ м§Җм •лҗҳм–ҙ лӢӨмқҢкіј к°ҷмқҖ мқ‘лӢөмқҙ мҳөлӢҲлӢӨ.

мӮ¬мҡ©мһҗ мқёмҰқмқ„ мІҳлҰ¬н•ҙлҙ…мӢңлӢӨ. (Authentication)

м§Җмӣҗн•ҳлҠ” мқёмҰқмқҳ мў…лҘҳ (rest_framework/authentication.py)

  • SessionAuthentication (мӣ№лёҢлқјмҡ°м Җм—җм„ң лЎңк·ёмқёмӢң м„ём…ҳм—җ лЎңк·ёмқёкё°лЎқмқ„ нҶөн•ҙ мқёмҰқ)
    • м„ём…ҳмқ„ нҶөн•ң мқёмҰқ м—¬л¶Җ мІҙнҒ¬
    • APIViewлҘј нҶөн•ҙ л””нҸҙнҠё м§Җм • (мҡ°м„ мҲңмң„ 1)
  • BasicAuthentication (httpмқҳ мқёмҰқн—ӨлҚ” мӮ¬мҡ©)
    • Basic мқёмҰқн—ӨлҚ”лҘј нҶөн•ң мқёмҰқ мҲҳн–ү (ex: Authorization: Basic YWxsaWV1czE6MTAyOXNoYWtl)
    • APIViewлҘј нҶөн•ҙ л””нҸҙнҠё м§Җм • (мҡ°м„ мҲңмң„ 2)
  • TokenAuthentication (apiм—җм„ң мЈјлЎң мӮ¬мҡ©)(djangoм—җм„ң мң м Җл§ҲлӢӨ 1:1лЎң tokenк°’к°Җм§җ,мһ¬л°ңкёүк°ҖлҠҘ)
    • Token н—ӨлҚ”лҘј нҶөн•ң мқёмҰқ мҲҳн–ү (ex: Authorization: Token 401f7ac837da42b97f613d789819ff93537bee6a)
  • RemoteUserAuthentication
    • User м •ліҙк°Җ лӢӨлҘё м„ңл№„мҠӨм—җм„ң кҙҖлҰ¬лҗ  л•Ң, Remote мқёмҰқ (мһҘкі  кіөмӢқл¬ём„ң)
    • Remote-User н—ӨлҚ”лҘј нҶөн•ң мқёмҰқ мҲҳн–ү

мқёмҰқ мІҳлҰ¬ мҲңм„ң

  1. л§Ө мҡ”мІӯ мӢңл§ҲлӢӨ APIViewмқҳ dispatch(request) нҳём¶ң
  2. APIViewмқҳ initial(request) нҳём¶ң
  3. APIViewмқҳ perform_authentication(request) нҳём¶ң
  4. Requestмқҳ user Property нҳём¶ң (self.request.user мӮ¬мҡ©)
  5. Requestмқҳ _authenticate() нҳём¶ң
    • APIViewлҘј нҶөн•ҙ м§Җм •лҗң Authentication нҳём¶ң

нҸ¬мҠӨнҢ…мқ„ м ҖмһҘн•  л•Ң, нҳ„мһ¬ мқёмҰқлҗң мң м Җ м •ліҙлҘј кё°лЎқнҶ лЎқ н•ҙліҙкІ мҠөлӢҲлӢӨ. authorн•„л“ңлҘј APIлҘј нҶөн•ҙ м§Җм •лҗҳм§Җ м•ҠлҸ„лЎқ PostSerializerмқҳ Meta.fieldsм—җм„ң authorн•„л“ңлҘј м ңмҷёмӢңмјңмЈјм„ёмҡ”.

мқҙм ң APIлҘј нҶөн•ҙ Post м ҖмһҘ мӢңм—җ нҳ„мһ¬ мқёмҰқлҗң мң м ҖлҘј м§Җм •н•ҙлҙ…мӢңлӢӨ.

perform_create н•ЁмҲҳлҠ” rest_framework/mixins.pyлӮҙ CreateModelMixin нҒҙлһҳмҠӨм—җм„ң createмӢңм—җ нҳём¶ңмқ„ н•ҳкі  мһҲмҠөлӢҲлӢӨ. мӣҗлһҳ кө¬нҳ„мқҖ лӢЁмҲңнһҲ serializer.save() мқҙм§Җл§Ң, л©”мҶҢл“ң мҳӨлІ„лқјмқҙл”©мқ„ нҶөн•ҙ м ҖмһҘн•  н•„л“ңлҘј м¶”к°ҖлЎң м§Җм •н•ҙмӨ¬мҠөлӢҲлӢӨ. serializer.save(**kwargs)н•ЁмҲҳлҠ” kwargs н•ӯлӘ©мқ„ нҶөн•ҙ, м¶”к°ҖлЎң м ҖмһҘн•  н•„л“ңлҘј м§Җм •н•  мҲҳ мһҲмҠөлӢҲлӢӨ.

лӢӨмқҢкіј к°ҷмқҙ мҡ”мІӯн•ҙліҙл©ҙ 500 м—җлҹ¬мқ‘лӢөмқ„ л°ӣмҠөлӢҲлӢӨ. API мҡ”мІӯ мӢңм—җ мқёмҰқм •ліҙлҘј м ңкіөн•ҳм§Җ м•Ҡм•ҳкё° л•Ңл¬ём—җ self.request.userм—җ AnonymouseUser мқёмҠӨн„ҙмҠӨк°Җ н• лӢ№лҗҳм–ҙ лӘЁлҚём ҖмһҘм—җ мӢӨнҢЁн•ң кІғмһ…лӢҲлӢӨ. мҷёлһҳнӮӨм—җлҠ” вҖңлӘЁлҚё к°қмІҙвҖқк°Җ м•„лӢҢ вҖңнҢҢмқҙмҚ¬ к°қмІҙвҖқлҠ” н• лӢ№н•  мҲҳ м—ҶмҠөлӢҲлӢӨ.(лЎңк·ёмқёмқҙ м•Ҳлҗҳм–ҙмһҲлӢӨл©ҙ userModel мқёмҠӨн„ҙмҠӨк°Җ м•„лӢҲлқј AnonymouseUserмқёмҠӨн„ҙмҠӨк°Җ н• лӢ№лҗҳлҜҖлЎң)

perform_create нҳём¶ң м „м—җ мқёмҰқм—¬л¶ҖлҘј мІҙнҒ¬н•ҳлҠ” кІғмқҙ н•„мҡ”н•ҳкІ мҠөлӢҲлӢӨл§Ң, мқҙлҠ” л’Өм—җм„ң мӮҙнҺҙліҙкІ мҠөлӢҲлӢӨ.

лӢӨмқҢ лӘ…л №мңјлЎң API мҡ”мІӯ мӢңм—җ HTTP Basic мқёмҰқн—ӨлҚ”лҘј м ңкіөн•ҙлҙ…мӢңлӢӨ. --auth мқёмһҗлЎң мӣ№лЎңк·ёмқё м•„мқҙл””/м•”нҳёлҘј м§Җм •н•ҙмЈјм„ёмҡ”.

мқҙ мҡ”мІӯм—җм„ң "username:password" л¬ёмһҗм—ҙмқҖ base64лЎң мқёмҪ”л”©лҗҳм–ҙ Authorization н—ӨлҚ”лЎң м „лӢ¬лҗҳкі , мқҙ н—ӨлҚ”лҘј BasicAuthorizationм—җм„ң мқём§Җн•ҳм—¬ мқёмҰқмқ„ мІҳлҰ¬н•©лӢҲлӢӨ. httpbin.orgлҘј нҶөн•ҙ н—ӨлҚ”лҘј нҷ•мқён•ҙлҙ…мӢңлӢӨ.

base64 мқёмҪ”л”©мқҖ нҢҢмқҙмҚ¬м—җм„ңлҠ” лӢӨмқҢкіј к°ҷмқҙ нҷ•мқён•ҙліҙмӢӨ мҲҳ мһҲмҠөлӢҲлӢӨ.

мӣ№лёҢлқјмҡ°м ҖлҘј нҶөн•ң API м ‘к·јм—җм„ң лЎңк·ёмқё/лЎңк·ём•„мӣғ м§Җмӣҗн•ҳкё°

django-rest-frameworkлҠ” мӣ№лёҢлқјмҡ°м ҖлҘј нҶөн•ң API м ‘к·јлҸ„ м§Җмӣҗн•ҙмЈјкё°м—җ, мӣ№лёҢлқјмҡ°м ҖлҘј нҶөн•ң лЎңк·ёмқё/лЎңк·ём•„мӣғлҸ„ м§Җмӣҗн•ҙмЈјкі  мһҲмҠөлӢҲлӢӨ. мқҙлҠ” authм•ұмқҳ login/logoutл·°лҘј к·ёлҢҖлЎң нҷңмҡ©н•ҳкі  мһҲмңјл©°, н…ңн”ҢлҰҝл§Ң rest_framework/login.htmlлЎң ліҖкІҪн•ҙм„ң м Ғмҡ©лҗҳм–ҙмһҲмҠөлӢҲлӢӨ.

мӣ№лёҢлқјмҡ°м ҖлЎң http://localhost:8000м—җ м ‘мҶҚн•ҙліҙмӢңл©ҙ, нҷ”л©ҙ мҡ°мғҒлӢЁм—җ 'Login'лІ„нҠјмқҙ м—ҶмҠөлӢҲлӢӨ. н”„лЎңм қнҠё/urls.pyм—җ лӢӨмқҢкіј к°ҷмқҙ URLConf м„Өм •мқ„ ліҖкІҪн•ҙмЈјм„ёмҡ”.

мқҙм ң LoginлІ„нҠјмқҙ ліҙм—¬м§‘лӢҲлӢӨ. нҒҙлҰӯн•ҙм„ң лЎңк·ёмқёмқ„ мҲҳн–үн•ҙліҙм„ёмҡ”. :D

к¶Ңн•ң (Permission) мӢңмҠӨн…ң - л№„мқёмҰқ мҡ”мІӯм—җ н•ңн•ҙм„ң мқҪкё°к¶Ңн•ңл§Ң л¶Җм—¬н•ҳкё°

django-rest-frameworkм—җм„ңлҠ” Permission мӢңмҠӨн…ңмқ„ м ңкіөн•ҙмЈјкі  мһҲмҠөлӢҲлӢӨ.

к¶Ңн•ң мІҙнҒ¬лҠ” лӢӨмқҢ кё°ліё лЈ°мқ„ к°Җм§‘лӢҲлӢӨ. (djangoл§ҢмңјлЎң м§„н–үн•ңлӢӨл©ҙ)

  • (userModelм—җм„ң н•„л“ңк°Җ) is_superuser=True мң м ҖлҠ” лі„лҸ„ Permissionмқ„ м§Җм •н•ҳм§Җ м•Ҡм•„лҸ„ лӘЁл“  к¶Ңн•ңмқҙ н—Ҳмҡ©
  • is_staff=True мң м ҖлҠ” /admin/ м ‘мҶҚл§Ң к°ҖлҠҘн•  лҝҗ, мқјл°ҳ мң м ҖмҷҖ лҸҷмқјн•ҳкІҢ н—Ҳмҡ©лҗң к¶Ңн•ңл§Ң к°ҖлҠҘ
  • is_active=False мң м ҖлҠ” к¶Ңн•ң м§Җм •м—¬л¶Җм—җ мғҒкҙҖм—Ҷмқҙ, лӘЁл“  к¶Ңн•ң л¶Ҳн—Ҳ (is_active=TrueмқёкІҪмҡ°м—җл§Ң лЎңк·ёмқёк°ҖлҠҘ)

django-rest-frameworkм—җм„ң кё°ліё м ңкіөн•ҳлҠ” PermissionлҠ” лӢӨмқҢкіј к°ҷмҠөлӢҲлӢӨ.

  1. AllowAny : мқёмҰқм—¬л¶Җм—җ мғҒкҙҖм—Ҷмқҙ, л·° нҳём¶ң н—Ҳмҡ© (л””нҸҙнҠё м§Җм •)
  2. IsAuthenticated : мқёмҰқлҗң мҡ”мІӯм—җ н•ңн•ҙм„ң, л·° нҳём¶ң н—Ҳмҡ©
  3. IsAdminUser : Staff мқёмҰқ мҡ”мІӯм—җ н•ңн•ҙм„ң, л·° нҳём¶ң н—Ҳмҡ©
  4. IsAuthenticatedOrReadOnly : л№„мқёмҰқ мҡ”мІӯм—җкІҢлҠ”, мқҪкё° к¶Ңн•ңл§Ң н—Ҳмҡ©
  5. DjangoModelPermissions : мқёмҰқлҗң мҡ”мІӯм—җ н•ңн•ҙм„ңл§Ң л·° нҳём¶ңмқ„ н—Ҳмҡ©н•ҳкі , м¶”к°ҖлЎң мң м Җлі„ мқёмҰқ к¶Ңн•ңмІҙнҒ¬лҘј мҲҳн–ү
  6. DjangoModelPermissionsOrAnonReadOnly : DjangoModelPermissionsкіј мң мӮ¬н•ҳлӮҳ, л№„мқёмҰқ мҡ”мІӯм—җ лҢҖн•ҙм„ңлҠ” мқҪкё° к¶Ңн•ңл§Ң н—Ҳмҡ©
  7. DjangoObjectPermissions
    • л№„мқёмҰқлҗң мҡ”мІӯмқҖ кұ°л¶Җ
    • мқёмҰқлҗң мҡ”мІӯм—җ н•ң, Record м ‘к·јм—җ лҢҖн•ң к¶Ңн•ңмІҙнҒ¬лҘј м¶”к°ҖлЎң мҲҳн–ү

APIView нҒҙлһҳмҠӨм—җм„ңлҠ” permission_classes мҶҚм„ұмқ„ нҶөн•ҙ APIлі„лЎң к¶Ңн•ң мІҙнҒ¬лҘј лӢӨлҘҙкІҢ к°Җм ёк°Ҳ мҲҳ мһҲмҠөлӢҲлӢӨ. ViewSetмқҖ APIViewлҘј мғҒмҶҚл°ӣм•ҳмңјлҜҖлЎң лҸҷмқјн•ҳкІҢ м§Җм •к°ҖлҠҘн•©лӢҲлӢӨ. лӢӨмқҢкіј к°ҷмқҙ IsAuthenticatedлҘј м§Җм •н•ҙліҙм„ёмҡ”.

мқҙм ң мқёмҰқлҗҳм§Җ м•ҠмқҖ лӘЁл“  мҡ”мІӯм—җ лҢҖн•ҙм„ңлҠ” лӘЁл‘җ кұ°л¶Җ 403 Forbidden мқ‘лӢөмқ„ л°ӣкІҢлҗ©лӢҲлӢӨ.

лӢӨмқҢкіј к°ҷмқҙ мқёмҰқм—җ нҶөкіјн•ҳл©ҙ, н•ҙлӢ№ APIмҡ”мІӯмқҖ м •мғҒм ҒмңјлЎң мІҳлҰ¬лҗ©лӢҲлӢӨ.

м»ӨмҠӨн…Җ Permission л§Ңл“Өкё°

django-rest-frameworkм—җм„ң кё°ліё м ңкіөн•ҙмЈјлҠ” Permissionл§ҢмңјлЎңлҸ„ лҢҖк°ң м¶©л¶„н•©лӢҲлӢӨл§Ң, н•„мҡ”м—җ мқҳн•ҙ м»ӨмҠӨн…Җ Permissionмқ„ л§Ңл“Өкі  мӢ¶мқ„ мҲҳ мһҲмҠөлӢҲлӢӨ.

лӘЁл“  Permission нҒҙлһҳмҠӨлҠ” лӢӨмқҢ 2к°Җм§Җ н•ЁмҲҳлҘј м„ нғқм ҒмңјлЎң кө¬нҳ„н•©лӢҲлӢӨ.

  • has_permission(request, view): л·° нҳём¶ң м ‘к·јк¶Ңн•ң

    • APIView м ‘к·ј мӢң, мІҙнҒ¬
    • мқҙлҘј кө¬нҳ„н•ң Permission нҒҙлһҳмҠӨ : IsAuthenticated, IsAdminUser, IsAuthenticatedOrReadOnly, DjangoModelPermissions, DjangoModelPermissionsOrAnonReadOnly

  • has_object_permission(request, view, obj): к°ңлі„ Record м ‘к·јк¶Ңн•ң

    • APIViewмқҳ get_objectн•ЁмҲҳлҘј нҶөн•ҙ object нҡҚл“қ мӢң, мІҙнҒ¬ - к°ңлі„ GET/PUT/DELETE мҡ”мІӯ
    • лёҢлқјмҡ°м ҖлҘј нҶөн•ң API м ‘к·јм—җм„ң CREATE/UPDATE Form л…ём¶ң м—¬л¶Җ нҷ•мқё мӢңм—җ, мІҙнҒ¬
    • мқҙлҘј кө¬нҳ„н•ң Permission нҒҙлһҳмҠӨ : DjangoObjectPermissions
