мӢңмһ‘н•ҳкё°м—җ м•һм„ң

мЈјмқҳ: мӢӨм ң м„ңл№„мҠӨм—җм„ңлҠ” id=Noneкіј к°ҷмқҖ мҪ”л“ңлҠ” м ҲлҢҖ м“°м§Җ л§Ҳм„ёмҡ”.

мёЎм • Metric

лӢӨмқҢ мҪ”л“ңлҘј нҶөн•ҙ, мҡ”мІӯ мІҳлҰ¬мӢңмқҳ к°Ғ л¶Җл¶„м—җ лҢҖн•ң мӢңк°„мқ„ мёЎм •н•ҙліҙкІ мҠөлӢҲлӢӨ.

  • Database Lookup (db_time) : лҚ°мқҙн„°лІ мқҙмҠӨ Fetch мҲҳн–үмӢңк°„(DBм—җм„ң к°Җм ёмҳӨлҠ”мӢңк°„)
  • Serialization (serializer_time) : Serializer м§Ғл ¬нҷ” мҲҳн–үмӢңк°„
  • API View (api_view_time) : APIView мҲҳн–үмӢңк°„(мқёмҰқкҙҖл ЁмІҳлҰ¬, нҺҳмқҙм§•мІҳлҰ¬л“ұл“ұ)
  • Response rendering (render_time) : Response л ҢлҚ”л§Ғ мҲҳн–үмӢңк°„
  • Django request/response : request/response мҲҳн–үмӢңк°„

totalмӢңк°„мқҖ мҡ”мІӯмқҙ мҡ”мІӯлҗҳкі  мқ‘лӢөмқҙ лӢӨмғқм„ұлҗҳкі  лӢ«кё°к№Ңм§Җмқҳ мӢңк°„

н•ҳм§Җл§Ң к°ңл°ңмғқмӮ°м„ұмқҙмһҲмңјлҜҖлЎң, SerializerлҘј м“°лҚ”лқјлҸ„ м Ғм Ҳн•ң мәҗмӢұмқ„ нҶөн•ҙм„ң м„ұлҠҘлҪ‘кё°к°ҖлҠҘ
